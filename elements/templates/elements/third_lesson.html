Функции

Функции позволяют организовать код в многократно используемые блоки.

1. Определение функции

def greet(name):
    return f"Hello, {name}!"

# Вызов функции
print(greet("Alice"))

2. Аргументы функции

Функции могут принимать произвольное количество аргументов:

def add(a, b):
    return a + b

print(add(5, 3))

Можно задать значения по умолчанию:

def greet(name, greeting="Hello"):
    return f"{greeting}, {name}!"

print(greet("Alice"))
print(greet("Bob", "Hi"))

3. Возвращение значений

Функция может возвращать несколько значений:

def get_person():
    name = "Alice"
    age = 25
    return name, age

name, age = get_person()
print(name, age)

Модули

Модуль — это файл с кодом на Python, который может содержать переменные, функции и классы.

1. Импорт модулей

import math

print(math.sqrt(16))

2. Импорт отдельных функций

from math import sqrt

print(sqrt(16))

3. Создание собственного модуля

Создайте файл mymodule.py:

# mymodule.py
def greet(name):
    return f"Hello, {name}!"

Импортируйте и используйте модуль:

import mymodule

print(mymodule.greet("Alice"))

Работа с файлами

Python предоставляет встроенные функции для работы с файлами.

1. Открытие и закрытие файла

# Открытие файла
file = open("example.txt", "w")

# Запись в файл
file.write("Hello, World!")

# Закрытие файла
file.close()

2. Чтение из файла

# Открытие файла для чтения
file = open("example.txt", "r")

# Чтение содержимого файла
content = file.read()

# Закрытие файла
file.close()

print(content)

3. Использование конструкции with

Конструкция with автоматически закрывает файл после работы с ним:

with open("example.txt", "w") as file:
    file.write("Hello, World!")

with open("example.txt", "r") as file:
    content = file.read()

print(content)
