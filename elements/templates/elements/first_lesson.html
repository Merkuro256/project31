Основы программирования на Python

Цели лекции:

  1. Ознакомиться с языком программирования Python.
  2. Узнать о ключевых особенностях Python.
  3. Изучить базовый синтаксис и основные конструкции языка.

Введение в Python

Python - это высокоуровневый, интерпретируемый язык программирования с динамической типизацией, разработанный Гвидо ван Россумом и впервые выпущенный в 1991 году. Он известен своей простотой и читаемостью кода, что делает его популярным среди начинающих программистов и профессионалов.

Основные особенности Python:

  1. Простота и читаемость: Код на Python часто выглядит как псевдокод, благодаря чему легко понимается и пишется.
  2. Богатая стандартная библиотека: Python имеет обширную стандартную библиотеку, которая поддерживает многие общие задачи программирования.
  3. Кроссплатформенность: Python работает на различных платформах, включая Windows, macOS и Linux.
  4. Активное сообщество: Широкое сообщество пользователей и разработчиков, множество доступных ресурсов и библиотек.

Установка Python

Перед началом работы необходимо установить Python на ваш компьютер.

  1. Перейдите на официальный сайт Python: python.org.
  2. Скачайте последнюю стабильную версию Python.
  3. Следуйте инструкциям по установке для вашей операционной системы (Windows, macOS, Linux).

После установки проверьте корректность установки, открыв командную строку (или терминал) и введя: 

python --version

или

python3 --version

Вы должны увидеть версию Python, установленную на вашем компьютере.

Основы синтаксиса Python

1. Вывод на экран

Для вывода текста на экран используется функция print(): 

print("Hello, World!")

2. Переменные и типы данных

Python поддерживает несколько основных типов данных: строки (str), целые числа (int), числа с плавающей точкой (float), булевы значения (bool) и другие. 

# Примеры переменных и их типов данных
name = "Alice"       # строка
age = 25             # целое число
height = 5.6         # число с плавающей точкой
is_student = True    # булево значение

3. Операторы

Python поддерживает основные арифметические операторы:

# Арифметические операторы
a = 10
b = 3
print(a + b)   # Сложение
print(a - b)   # Вычитание
print(a * b)   # Умножение
print(a / b)   # Деление
print(a % b)   # Остаток от деления
print(a ** b)  # Возведение в степень

4. Условные операторы

Условные операторы позволяют выполнять различные действия в зависимости от условия: 

# Пример условного оператора
x = 10
if x > 5:
    print("x больше 5")
elif x == 5:
    print("x равно 5")
else:
    print("x меньше 5")

5. Циклы

Циклы позволяют повторять выполнение кода несколько раз:

# Цикл for
for i in range(5):
    print(i)

# Цикл while
count = 0
while count < 5:
    print(count)
    count += 1

6. Функции

Функции позволяют организовывать код в многократно используемые блоки: 

# Определение функции
def greet(name):
    print("Hello, " + name + "!")

# Вызов функции
greet("Alice")
