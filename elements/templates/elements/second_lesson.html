Переменные и типы данных

В предыдущей лекции мы познакомились с переменными и базовыми типами данных. Сегодня мы рассмотрим их подробнее.

1. Переменные

Переменные используются для хранения данных. В Python переменные создаются при присваивании им значения:

name = "Alice"
age = 25
height = 5.6
is_student = True

Переменные могут менять свой тип в процессе выполнения программы:

value = 10      # int
value = "text"  # str

2. Типы данных

Python поддерживает несколько встроенных типов данных:

  • Строки (str): Хранение текстовой информации.
  • Целые числа (int): Хранение целых чисел.
  • Числа с плавающей точкой (float): Хранение вещественных чисел.
  • Булевы значения (bool): Логические значения True или False.

Для проверки типа данных переменной используется функция type():

print(type(name))    # 
print(type(age))     # 
print(type(height))  # 
print(type(is_student))  #

Структуры данных

Структуры данных - это специализированные форматы для организации, управления и хранения данных. В Python существуют следующие основные структуры данных:

1. Списки (lists)

Список - это упорядоченная изменяемая коллекция элементов:

# Создание списка
fruits = ["apple", "banana", "cherry"]

# Доступ к элементам списка
print(fruits[0])  # apple

# Изменение элемента
fruits[1] = "blueberry"

# Добавление элемента
fruits.append("orange")

# Удаление элемента
fruits.remove("cherry")

# Перебор элементов списка
for fruit in fruits:
    print(fruit)

2. Кортежи (tuples)

Кортеж - это упорядоченная неизменяемая коллекция элементов:

# Создание кортежа
coordinates = (10, 20)

# Доступ к элементам кортежа
print(coordinates[0])  # 10

# Кортежи неизменяемы, поэтому элементы изменить нельзя
# coordinates[0] = 15  # Ошибка

# Перебор элементов кортежа
for coord in coordinates:
    print(coord)

3. Множества (sets)

Множество - это неупорядоченная коллекция уникальных элементов:

# Создание множества
numbers = {1, 2, 3, 4, 5}

# Добавление элемента
numbers.add(6)

# Удаление элемента
numbers.remove(3)

# Перебор элементов множества
for number in numbers:
    print(number)

4. Словари (dictionaries)

Словарь - это неупорядоченная коллекция пар "ключ-значение":

# Создание словаря
student = {
    "name": "Alice",
    "age": 25,
    "courses": ["Math", "CompSci"]
}

# Доступ к значениям по ключам
print(student["name"])    # Alice

# Изменение значения
student["age"] = 26

# Добавление новой пары
student["grade"] = "A"

# Удаление пары
del student["courses"]

# Перебор элементов словаря
for key, value in student.items():
    print(key, value)
